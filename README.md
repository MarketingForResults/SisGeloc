# SisGeloc — Sistema de Gestão de Locação de Veículos

## Estrutura do Projeto
- `apps/api` — Backend Node.js + Express + TypeScript
- `apps/web` — Frontend React 18 + Vite + Tailwind
- `apps/mobile` — App React Native + Expo
- `packages/shared` — Tipos e utilitários compartilhados

## Como rodar
```bash
npm install
npm run dev:api
npm run dev:web
```
